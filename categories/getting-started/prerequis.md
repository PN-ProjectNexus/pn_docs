---
title: Prérequis
---

Avant d'installer les scripts PN Project Nexus, assurez-vous que votre serveur répond aux exigences suivantes.

## Infrastructure

- **Serveur FiveM** — Artifacts recommandés : **latest** (build 5504+)
- **Base de données** — MySQL / MariaDB (5.7+ ou 10.3+)
- **Framework** — ESX Legacy 1.9+ **ou** QBCore

## Dépendances obligatoires

| Resource | Utilité |
|----------|---------|
| `oxmysql` | Connexion base de données |
| `ox_lib` | Fonctions utilitaires et notifications |
| `es_extended` / `qb-core` | Framework selon votre choix |

## Dépendances optionnelles

| Resource | Utilité |
|----------|---------|
| `ox_inventory` | Gestion d'inventaire avancée |
| `ox_target` | Menu contextuel ciblé |
| `ox_doorlock` | Gestion des portes |
| `interact-sound` | Sons spatialisés |

## Connaissances recommandées

- Savoir éditer des fichiers `.lua` et `.json`
- Connaître la structure d'un artifact FiveM
- Savoir configurer un serveur via `server.cfg`
- Bases de Git pour suivre les mises à jour
